# Quickstart: Market Dashboard

## Prerequisites
- Node.js 18+
- npm

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

## Development
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open [http://localhost:3000](http://localhost:3000)

## Data Source
- By default, the app uses **Mock Data**.
- No API keys are required for the MVP.
